# Piranha-Extension
This open-source extension will allow users to block websites and report them to a community blacklist.
This project was developed by seniors in the computer science department at Tennessee State University for their senior project.
//
To setup the extension in your chrome browser, perform the following steps:
1. Download or clone the code from this repository to a local directory on your computer.
2. Go to Google Chrome and enter "chrome://extensions/" into the URL or click the puzzle piece on the top-right toolbar.
3. Toggle the switch to turn developer mode on in the top right of the page.
4. Select "Load Unpacked" on the left.
5. Select the directory you saved/cloned the repository to.
6. Remember to enable the extension by toggle the switch 
7. You should be ready to use the extension! Happy Reporting!
